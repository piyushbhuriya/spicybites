# Spicy Bites

Spicy Bites is a responsive restaurant website created using HTML, CSS, and JavaScript.

## Features

- Responsive Design
- Navigation Bar
- Hero Section with Background Image
- Restaurant Menu Image
- Food Order Cards
- Add to Cart Buttons
- Mobile Friendly Layout

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Description

This project is a simple restaurant website for beginners.  
It includes a food menu section, order items, and attractive UI design with images.
